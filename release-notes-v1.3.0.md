# Changes

- Refined the Mod Settings UI
- Added a new Move HUD global button in the main Mod Settings view
- Added a new Pets HUD overlay, showing your active pet(s) info.
- Added Static Gang/Truce Ping setting, allowing you to keep the ping in place without following you.
- Increased the duration setting of the gang/truce ping to 15 seconds (up from 10).
- Fixed the Following Gang/Truce Ping covering your screen with the beacon effect.
- Added feature "Clickable Coordinate Pings" created by Boston54 to allow for clickable coordinates in chat.
- Added feature "Add category key for pings in language file" created by RSNFreud

Thank you to the two contributors to the open source project! Remember, if you want to add something to the mod, you can do so to be reviewed.
